# Cases

`cases/` 目录用于承载不同公司规模下的模型工程案例层。

它不是额外的主题，而是和所有能力线正交的一层约束说明。

当前分为三类:

- [bigtech/README.md](/Volumes/ExtaData/newcode/llm-engineering-lab/cases/bigtech/README.md)
- [mid_company/README.md](/Volumes/ExtaData/newcode/llm-engineering-lab/cases/mid_company/README.md)
- [small_team/README.md](/Volumes/ExtaData/newcode/llm-engineering-lab/cases/small_team/README.md)

用途:

- 帮助读者理解为什么同一种技术路线，在不同团队里会有完全不同的最优解
- 避免把大厂做法不加区分地套到所有团队
- 把“工程取舍”变成显式内容，而不是隐含前提
