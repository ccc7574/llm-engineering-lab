# Tracks

`tracks/` 目录用于承载能力线级别的设计，而不是具体单个任务。

当前 V2 共有 6 条能力线:

- [pretraining/README.md](/Volumes/ExtaData/newcode/llm-engineering-lab/tracks/pretraining/README.md)
- [post_training/README.md](/Volumes/ExtaData/newcode/llm-engineering-lab/tracks/post_training/README.md)
- [coding/README.md](/Volumes/ExtaData/newcode/llm-engineering-lab/tracks/coding/README.md)
- [multimodal/README.md](/Volumes/ExtaData/newcode/llm-engineering-lab/tracks/multimodal/README.md)
- [agentic/README.md](/Volumes/ExtaData/newcode/llm-engineering-lab/tracks/agentic/README.md)
- [harness/README.md](/Volumes/ExtaData/newcode/llm-engineering-lab/tracks/harness/README.md)

使用方式:

- `tracks/` 负责定义一条能力线要学什么、为什么学、如何分层
- `tasks/` 负责把能力线拆成具体工单
- `cases/` 负责说明不同公司规模下的工程约束和取舍
